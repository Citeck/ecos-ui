# Глобальные иконки

Папка `global` содержит *несистемные* иконки, которые доступны не только для локальной разработки. Иконки, помещённые сюда, автоматически становятся доступны пользователям в интерфейсе выбора иконок (например, при выборе иконки для пунктов меню).

---

## 📁 Что должно находиться в `global`

* Иконки, которые должны быть доступны как **для локальной разработки**, так и **конечным пользователям** через интерфейс.
* Универсальные, часто используемые иконки, которые могут появляться в меню, на панелях инструментов и в других элементах интерфейса.

---

## Как это работает

* Любой файл иконки, помещённый в эту папку, автоматически попадает в **список иконок, доступных для выбора пользователем**.
* Если вы **не хотите**, чтобы иконка отображалась в пользовательском списке, переместите её уровнем выше — в `../icons`. Иконки в `../icons` доступны только для локальной разработки и **не показываются пользователю**.

---

## Сортировка иконок в пользовательском списке

Порядок отображения иконок можно регулировать с помощью параметра `weight`. Значение `weight` определяет индекс или приоритет иконки — чем больше число, тем ниже в списке она появится.

**По умолчанию** каждая иконка имеет очень большой вес, поэтому новые иконки оказываются в конце списка.

**Чтобы присвоить иконке собственный вес**, добавьте в модуль с иконкой следующую строку:

```js
// Устанавливает пользовательский вес. Большие числа помещают иконку ниже в списке.
export const weight = 0; // вместо 0 может быть любое число
```

Примечания:

* Чем выше значение, тем дальше вниз в списке попадёт иконка.
* Чем ниже значение (включая отрицательные), тем выше она будет отображаться.

---

## Рекомендации и лучшие практики

* Используйте `global` для стабильных, универсальных иконок, которые могут понадобиться многим пользователям.
* Специализированные или экспериментальные иконки держите в `../icons`, чтобы не перегружать пользовательский список.
* Настраивая `weight`, учитывайте важность и частоту использования иконки — наиболее востребованные иконки должны иметь меньший вес.

---

## Диагностика

Если иконка не отображается в интерфейсе, убедитесь, что:

1. Файл иконки действительно находится в папке `global`.
2. Файл корректно экспортирует иконку.
3. Нет ошибок сборки или импорта, которые могли бы помешать регистрации иконки.
