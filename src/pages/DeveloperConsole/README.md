# Developer Console

Консоль разработчика для администраторов позволяет выполнять JavaScript код прямо в браузере с отображением результатов.

## Возможности

- ✅ **Редактор кода** с синтаксической подсветкой (Monaco Editor)
- ✅ **Выполнение JS кода** в контексте браузера
- ✅ **Захват консоли**: Все log, warn, error, info выводятся в панель результатов
- ✅ **Отображение возвращаемых значений**
- ✅ **Плавающая панель результатов** с функцией перемещения и изменения размера
- ✅ **Индикатор загрузки** при выполнении кода
- ✅ **Полная поддержка async/await**
- ✅ **Сохранение состояния** в localStorage:
  - Код автоматически сохраняется при изменении
  - Размер панели результатов запоминается
  - Позиция панели (внизу или справа) сохраняется
- ✅ **Ресайз панели результатов** (перетаскивание границы)
- ✅ **Переключение позиции** панели (внизу ↔ справа)

## Доступ

- Открыть меню (в верхней части страницы)
- Нажать на "Developer Console" (доступно только для администраторов)
- URL: `/v2/developer-console`

## Использование

### Базовый пример
```javascript
// Выведет сообщение в консоль
console.log('Hello, World!');

// Вернет значение
42
```

### Работа с объектами
```javascript
const obj = { name: 'Test', value: 123 };
console.log(obj);
return obj;
```

### Асинхронный код
```javascript
const data = await fetch('/api/endpoint').then(r => r.json());
console.log(data);
return data;
```

## Компоненты

### DeveloperConsole.jsx
Основной компонент страницы консоли. Управляет состоянием кода, выполнением и отображением результатов.

### CodeEditor.jsx
Обертка вокруг Monaco Editor с поддержкой различных языков программирования.

### ResponsePanel.jsx
Плавающая панель результатов с функциями:
- Перемещение (drag-and-drop по заголовку)
- Изменение размера (из правого нижнего угла)
- Отображение логов консоли
- Отображение возвращаемых значений
- Отображение ошибок

## Стиль

Все компоненты используют темную тему VS Code для лучшей визуальной консистентности с редактором кода.

## Безопасность

⚠️ **Важно**: Эта функция доступна только администраторам и должна использоваться с осторожностью, так как позволяет выполнять произвольный JavaScript код.
